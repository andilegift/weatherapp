# 🌤️ WeatherApp

A sleek, responsive weather application that provides real-time forecasts, interactive maps, and insightful weather data.


## ✨ Features

- **Real-time Weather:** Get current conditions, temperature, humidity, wind speed, and more.
- **5-Day Forecast:** Plan ahead with a detailed multi-day forecast.
- **Interactive Map:** Visualize weather patterns using Leaflet.js with multiple data layers (Temperature, Precipitation, etc.).
- **Air Quality Index:** Check the current AQI for your location.
- **Smart Features:** Location detection, voice search, unit conversion (°C/°F), and recent search history.
- **User-Friendly UI:** Clean, modern design with a dark/light mode toggle.
- **Weather Alerts & Tips:** Receive important alerts and personalized weather advice.


## 🛠️ Built With

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **API:** [OpenWeatherMap API](https://openweathermap.org/api)
- **Maps:** [Leaflet.js](https://leafletjs.com/)
- **Icons:** [Font Awesome](https://fontawesome.com/)
- **Fonts:** [Google Fonts (Poppins)](https://fonts.google.com/specimen/Poppins)

## 📦 Installation

To run this project locally:

1.  Clone the repo:
    ```bash
    git clone https://github.com/YourUsername/weatherapp.git
    ```
2.  Open `index.html` in your browser.

**Note:** You will need your own API key from [OpenWeatherMap](https://openweathermap.org/api). Replace the `apiKey` variable in `script.js` with your key.
